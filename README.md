# 구현 기능 목록
## 기능 입력
**1. [입력] 기능을 입력받는다.**
   - [예외 처리] 주어진 기능이 아닌 경우 예외를 발생시킨다.
   - [예외 처리] 주말 또는 공휴일에 출석 확인 또는 수정 기능을 선택하는 경우 예외를 발생시킨다.
   - 
## 출석 확인
**1. [입력] 닉네임을 입력받는다.**
   - [예외 처리] 등록되지 않은 닉네임일 경우 예외를 발생시킨다.

**2. [입력] 등교 시간을 입력받는다.**
   - [예외 처리] 날짜 또는 시간을 잘못된 형식으로 입력한 경우 예외를 발생시킨다.
   - [예외 처리] 등교 시간이 캠퍼스 운영 시간이 아닌 경우 예외를 발생시킨다.

**3. [중간 과정] 출석 일시를 기록한다.**
   - [예외 처리] 이미 출석을 한 경우 예외를 발생시킨다.

**4. [출력] 출석 기록을 출력한다.**

## 출석 수정
**1. [입력] 닉네임을 입력받는다.**
   - [예외 처리] 등록되지 않은 닉네임일 경우 예외를 발생시킨다.

**2. [입력] 수정하려는 날짜(일)를 입력받는다.**
   - [예외 처리] 정수 타입으로 입력하지 않은 경우 예외를 발생시킨다.
   - [예외 처리] 수정하려는 날짜가 주말 또는 공휴일인 경우 예외를 발생시킨다.
   - [예외 처리] 미래 날짜를 입력한 경우 예외를 발생시킨다.

**3. [입력] 변경할 시간을 입력받는다.**
   - [예외 처리] 잘못된 형식으로 입력한 경우 예외를 발생시킨다.
   - [ㅖ외 처리] 캠퍼스 운영 시간이 아닌 경우 예외를 발생시킨다.

**4. [중간 과정] 입력받은 날짜의 출석 기록을 수정한다.**

**5. [출력] 수정 완료 메시지를 출력한다.**

## 크루별 출석 기록 확인
**1. [입력] 닉네임을 입력받는다.**\

**2. [중간 과정] 경고, 면담, 제적 대상자 여부를 체크한다.**
   
**3. [출력] 출석 기록을 출력한다.**

## 제적 위험자 확인
**1. [출력] 제적 위험자 조회 결과를 출력한다.**
